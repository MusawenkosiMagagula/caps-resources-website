# CAPS Resources - Online Store

A professional, vibrant online store for selling CAPS-aligned educational PDF resources from Preschool to Grade 12.

## Features

- **Vibrant Design**: Colorful grade cards with unique colors for each level
- **Simple UI**: Clean, intuitive interface inspired by IXL Learning
- **Grade Navigation**: Easy browsing from Preschool to Grade 12
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Instant Downloads**: PDF resources delivered immediately after purchase
- **Shopping Cart**: Built-in cart functionality
- **Smooth Animations**: Professional animations and transitions

## How to Use

1. Open `index.html` in any modern web browser
2. Browse through the different grade levels
3. Click "View Resources" on any grade card to see available resources
4. Add items to cart (functionality ready for backend integration)
